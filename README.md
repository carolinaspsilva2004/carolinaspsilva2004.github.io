# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, developed to present academic and personal projects throughout my academic and professional journey in Software Engineering.

The portfolio serves as a centralized platform to showcase projects across multiple domains, including Machine Learning, Software Engineering, Full-Stack Development, Mobile Computing, and Data Visualization, with a strong emphasis on clean design, usability, and modern web development practices.

## Overview

The main objective of this project is to provide a clear and structured overview of my technical background, skills, and experience. In addition to presenting individual projects, the website reflects my approach to software engineering, focusing on maintainability, scalability, and user-centered design.

---

## Key Features

- **Project Showcase**  
  Structured presentation of academic and personal projects, including detailed descriptions, technologies used, and links to source repositories.

- **Interactive Project Filtering**  
  Dynamic filtering system allowing users to navigate projects by domain and technical focus.

- **Responsive and Modern User Interface**  
  Clean, card-based layout designed with responsiveness and accessibility in mind.

- **Integrated Modals and Navigation**  
  Modal-based interactions for CV download and external links (GitHub, LinkedIn, contact).

- **Enhanced User Experience**  
  Smooth transitions, hover states, and subtle animations to improve usability and visual clarity.

---

## Technology Stack

- **Frontend:** React, TypeScript  
- **Styling:** Custom CSS with design tokens  
- **Routing:** React Router  
- **Build Tool:** Vite  
- **Version Control:** Git and GitHub
